# 네이버 검색 등록 — 10분 퀵스타트 (www.mhj.nz)

> 그대로 따라 하시면 됩니다. 상세 설명·구글/빙 병행은 [naver-search-advisor-setup.md](naver-search-advisor-setup.md) 참고.
> 이 사이트는 소유권 확인 태그가 이미 심어져 있어 **확인 단계가 클릭 한 번에 통과**됩니다.

## 준비물
- 네이버 계정 (로그인용)

## 체크리스트

- [ ] **1. 접속·로그인** — https://searchadvisor.naver.com → 네이버 로그인 → 상단 **웹마스터 도구**

- [ ] **2. 사이트 등록** — **사이트 관리 → 사이트 등록** → 아래 주소 입력 후 등록
  ```
  https://www.mhj.nz
  ```
  ⚠️ `https` + `www` 정확히, 끝 슬래시(/) 없이. (네이버는 http/https·www 유무를 다른 사이트로 취급)

- [ ] **3. 소유권 확인** — 확인 방법에서 **"HTML 태그"** 선택 → **확인** 클릭
  - 이미 심어둔 값과 같아 즉시 통과됩니다.
  - 만약 네이버가 **다른 코드**를 발급하면 → 그 코드를 저에게 알려주세요(교체·배포 후 다시 확인).

- [ ] **4. RSS 제출** — 좌측 **요청 → RSS 제출** → 아래 입력
  ```
  feed.xml
  ```

- [ ] **5. 사이트맵 제출 (가장 중요)** — 좌측 **요청 → 사이트맵 제출** → 아래 입력
  ```
  sitemap.xml
  ```

- [ ] **6. 핵심 페이지 수집 요청 (색인 가속)** — 좌측 **요청 → 웹페이지 수집** → 아래 URL을 하나씩 제출
  ```
  https://www.mhj.nz/
  https://www.mhj.nz/blog
  https://www.mhj.nz/magazine
  https://www.mhj.nz/about
  ```

- [ ] **7. 진단 확인** — **사이트 진단**에서 robots.txt·수집 현황 오류 없는지 확인

## 끝! 이후
- 네이버 색인까지 **통상 1~4주**. (사이트맵 + 수집요청 병행 시 단축)
- 막히거나 새 코드를 요구하면 → 저에게 알려주세요. 바로 이어받겠습니다.
