# 1. 판형 & 물리적 규격

## 왜 이 판형인가

세계적 라이프스타일 매거진의 실제 판형:

| 매거진 | 판형 (mm) | 비율 | 비고 |
|--------|-----------|------|------|
| **Kinfolk** | 275 × 215 | 1.28:1 | MHJ의 1차 벤치마크 |
| **Monocle** | 265 × 200 | 1.33:1 | 컴팩트 프리미엄 |
| **Cereal** | ~250 × 200 | 1.25:1 | 미니멀 에디토리얼 |
| Harper's Bazaar | 290 × 215 | 1.35:1 | 패션 스탠다드 |
| Vogue UK | 290 × 210 | 1.38:1 | A4에 가까움 |
| A4 (참고) | 297 × 210 | 1.41:1 | ISO 표준 |

## MHJ 확정 판형

```
┌─────────────────────────────────────┐
│        MHJ TRIM SIZE                │
│                                     │
│        Width:  210 mm               │
│        Height: 275 mm               │
│        Ratio:  1.31:1               │
│                                     │
│        디지털 렌더링:               │
│        2480 × 3248 px (300dpi)      │
│        1240 × 1624 px (150dpi/web)  │
│                                     │
│        CSS 비율: 210 / 275 = 0.7636 │
│        → aspect-ratio: 42 / 55      │
│                                     │
└─────────────────────────────────────┘
```

**선정 이유:**
- Kinfolk(275×215)과 Monocle(265×200)의 중간 — MHJ의 "슬로우 라이프 에디토리얼" 정체성에 적합
- 높이 275mm: Kinfolk과 동일. 세로 여유 충분, 사진 임팩트 보장
- 너비 210mm: A4 너비와 동일. NZ/호주 인쇄소에서 A4 원지에 최적화 가능 → 인쇄 비용 절감
- "A4보다 약간 짧은 프리미엄 매거진" — 서가에서 일반 서류와 구분되는 고급감

## 영역 정의 (인쇄 대비)

```
┌─ Bleed (3mm 사방) ─────────────────────────┐
│  ┌─ Trim (210 × 275mm) ──────────────────┐  │
│  │  ┌─ Live Area / Safe Zone ──────────┐  │  │
│  │  │                                   │  │  │
│  │  │   Top margin:    18mm             │  │  │
│  │  │   Bottom margin: 20mm             │  │  │
│  │  │   Outside margin: 15mm            │  │  │
│  │  │   Inside margin:  18mm (gutter)   │  │  │
│  │  │                                   │  │  │
│  │  │   Live area:                      │  │  │
│  │  │   177 × 237mm                     │  │  │
│  │  │                                   │  │  │
│  │  └───────────────────────────────────┘  │  │
│  └────────────────────────────────────────┘  │
└──────────────────────────────────────────────┘
```

| 영역 | 크기 | 용도 |
|------|------|------|
| **Bleed** | Trim + 3mm 사방 = 216 × 281mm | 인쇄 시 재단 여유. 풀 블리드 사진은 여기까지 |
| **Trim** | 210 × 275mm | 최종 완성 크기 |
| **Live Area** | 177 × 237mm | 모든 텍스트와 핵심 이미지는 반드시 이 안에 |
| **Gutter** | Inside 18mm | 제본 시 접히는 부분. 텍스트 절대 금지 |

## 디지털 렌더링 매핑

웹에서는 고정 px 대신 비율 기반으로 렌더링:

```css
.magazine-page {
  aspect-ratio: 42 / 55;          /* 210:275 정확 비율 */
  max-width: 620px;               /* 뷰어 최대 너비 */
  margin: 0 auto;
  background: var(--page-bg);
  overflow: hidden;
  position: relative;
}

.magazine-page-inner {
  /* Live area = padding으로 구현 */
  padding: 6.55% 7.14% 7.27% 8.57%;
  /* top:18/275, right:15/210, bottom:20/275, left:18/210 */
}
```

# 2. 그리드 시스템

## 왜 12-column인가

Kinfolk은 12-column 그리드를 사용하며, 이는 레이아웃에 더 많은 유연성을 제공하고 매거진 전체의 속도감을 변화시킬 수 있게 한다.

12는 2, 3, 4, 6으로 나눌 수 있어 가장 유연한 그리드 단위다. MHJ도 12-column을 채택한다.

## MHJ 12-Column Grid

```
Live Area 너비: 177mm
Column 수: 12
Gutter 너비: 3mm (column 사이)
Column 너비: (177 - 11×3) / 12 = 12mm

┌──────────────── 177mm (Live Area) ────────────────┐
│ C1 │g│ C2 │g│ C3 │g│ C4 │g│ C5 │g│ C6 │g│...│C12│
│12mm│3│12mm│3│12mm│3│12mm│3│12mm│3│12mm│3│   │    │
└───────────────────────────────────────────────────┘

실용 조합:
├── 6+6 ───────┤  2단 균등 (에세이, 인터뷰)
├── 4+4+4 ─────┤  3단 균등 (목차, 디렉토리)
├── 8+4 ───────┤  비대칭 (본문 넓게 + 사이드바)
├── 5+7 ───────┤  비대칭 (사진 + 텍스트)
├── 3+9 ───────┤  극단 비대칭 (캡션 + 대형 사진)
├── 12 ────────┤  전폭 (풀 블리드 사진, 타이틀)
```

## Baseline Grid

```
본문 기준: Noto Sans KR 9pt / leading 14pt
Baseline grid: 14pt (≈ 4.94mm) 간격
모든 텍스트 프레임은 baseline grid에 정렬

→ 세로 방향의 리듬감 유지
→ 좌우 페이지 텍스트 줄이 정확히 일치
→ 사진 높이도 baseline grid의 배수로 맞춤
```

## 수평 모듈

```
Live Area 높이: 237mm
Baseline grid: ~5mm
수평 모듈: 8등분 = 29.6mm (≈6줄)

이 모듈 조합으로 사진/텍스트 영역 높이를 결정:
- 2모듈(59mm): 작은 사진, 인용구
- 3모듈(89mm): 중간 사진
- 4모듈(119mm): 대형 사진, 반 페이지
- 6모듈(178mm): 거의 전면 사진
- 8모듈(237mm): 풀 페이지 (Live area 전체)
```

# 3. 타이포그래피 체계

## 서체 팔레트 (3 Family Rule)

세계적 매거진은 서체를 3개 패밀리 이내로 제한한다. MHJ도 이 원칙을 따른다.

| 역할 | 서체 | Weight | 용도 |
|------|------|--------|------|
| **Display** | Playfair Display | 400, 700, 900, Italic | 표지 제목, 섹션 오프너, 대형 제목 |
| **Body (EN)** | Inter | 400, 500, 600 | 영문 본문, UI 라벨, 캡션 |
| **Body (KR)** | Noto Sans KR | 400, 500, 700, 900 | 한국어 본문, 라벨 |

> **왜 Inter인가?** 기존 MHJ 웹사이트와의 일관성. 매거진 본문에서도 가독성이 뛰어남. 향후 커스텀 서체로 교체 가능하나, Phase 1에서는 Google Fonts로 통일.

## 타입 스케일 (Type Scale)

모든 크기는 **인쇄 기준 pt** 단위로 정의. 디지털 렌더링 시 px 변환 적용.

### 표지 (Cover)

| 요소 | 서체 | Size | Weight | Tracking | 비고 |
|------|------|------|--------|----------|------|
| Masthead "MHJ" | Playfair Display | 72pt | 400 | +20 | 항상 동일 위치 |
| Cover Title | Playfair Display | 36–48pt | 900 | 0 | 최대 2줄 |
| Cover Subtitle | Inter | 11pt | 500 | +50 | uppercase |
| Issue Info | Inter | 8pt | 400 | +100 | "ISSUE 08 · WINTER 2026" |

### 내지 (Interior)

| 요소 | 서체 | Size | Weight | Leading | Tracking |
|------|------|------|--------|---------|----------|
| **Section Title** | Playfair Display | 36–48pt | 900 | 110% | 0 |
| **Article Title** | Playfair Display | 24–32pt | 700 | 115% | -10 |
| **Standfirst** (리드 문장) | Inter | 12pt | 400 italic | 150% | 0 |
| **Body Text** | Noto Sans KR / Inter | 9pt | 400 | 155% (14pt leading) | +5 |
| **Pull Quote** | Playfair Display Italic | 18–24pt | 400 | 130% | 0 |
| **Caption** | Inter | 7.5pt | 400 | 135% | +10 |
| **Kicker** (카테고리 라벨) | Inter | 7pt | 600 | — | +150, uppercase |
| **Byline** | Inter | 8pt | 500 | — | +30 |
| **Page Number** | Inter | 7pt | 400 | — | +50 |
| **Running Header** | Inter | 6.5pt | 500 | — | +200, uppercase |
| **Sidebar Title** | Noto Sans KR | 11pt | 700 | 130% | 0 |
| **Sidebar Body** | Noto Sans KR | 8pt | 400 | 150% | +5 |

### 본문 가독성 규칙

```
- 행 길이 (measure): 50~70자/행 (영문 기준)
  → 12-column grid에서 6~8 column 너비가 이상적
- 영문 본문: Inter 9pt / 14pt leading = 1.56 비율
- 한국어 본문: Noto Sans KR 9pt / 14pt leading
- 단락 구분: 들여쓰기 1em (첫 단락 제외) 또는 행간 1줄
- 하이픈: 영문 자동 하이프네이션 ON
- Widow/Orphan: 절대 금지 (마지막 줄에 단어 1개만 남기면 안 됨)
```

# 4. 컬러 시스템

## 기본 원칙: Achromatic + One Accent

MHJ는 Kinfolk/Cereal 스타일의 **절제된 색채**를 사용한다. 기본은 무채색이고, 악센트는 단 하나.

## 매거진 컬러 팔레트

### Core (항상 사용)

| 이름 | Hex | 용도 |
|------|-----|------|
| **Page White** | `#FDFCFA` | 페이지 배경. 순백(#FFF)보다 따뜻함 |
| **Ink Black** | `#1A1A1A` | 본문 텍스트 |
| **Deep Brown** | `#8A6B4F` | 브랜드 컬러. Masthead, 구분선, 강조 |
| **Warm Gray** | `#9B9590` | 캡션, 페이지 넘버, 보조 텍스트 |
| **Cream** | `#FAF8F5` | 사이드바 배경, 인용구 배경 |
| **Light Border** | `#EDE9E3` | 구분선, 사진 테두리 (필요 시) |

### Accent (이슈별 1색 선택)

매 이슈마다 계절/주제에 맞는 악센트 1색을 선택한다. 이 색은 해당 이슈 전체에 일관되게 사용.

| 시즌 | 추천 악센트 | Hex | 용도 |
|------|------------|-----|------|
| Spring | Sage Green | `#7D8E7F` | 키커, 풀쿼트 구분선, 드롭캡 |
| Summer | Ocean Blue | `#5B7E95` | 〃 |
| Autumn | Burnt Sienna | `#C17B4A` | 〃 |
| Winter | Slate Blue | `#6B7B8D` | 〃 |

### 금지 색

```
❌ 인디고/퍼플 (#4F46E5) — 웹사이트 AI Insight 전용. 매거진에서 절대 사용 금지
❌ 순수 검정 (#000000) — 항상 #1A1A1A 사용
❌ 순수 백색 (#FFFFFF) — 항상 #FDFCFA 사용
❌ 채도 높은 원색 — 파스텔/뮤트 톤만 허용
```

## 사진 위 텍스트 컬러

```
밝은 사진 위: #1A1A1A (Ink Black)
어두운 사진 위: #FDFCFA (Page White) + text-shadow 미세 적용
중간 톤 사진 위: 반투명 오버레이 깔기 (rgba(253,252,250,0.85))

→ 가독성이 최우선. 예쁨보다 읽힘이 먼저.
```

# 5. 사진 & 이미지 규칙

## 사진 톤 가이드 (MHJ Warm)

MHJ 매거진의 사진은 Kinfolk의 "quiet, intimate" 톤을 따른다.

```
Lightroom 프리셋 기준값:
- 채도 (Saturation): -15 ~ -20
- 하이라이트 온도 (Highlight Temp): +10 ~ +15
- 섀도 리프트 (Shadow Lift): +15 ~ +20  
- 선명도 (Clarity): +5 ~ +10
- 비네팅 (Vignette): -10 ~ -15
- 톤 커브: S-curve 완만하게 (콘트라스트 약간 올림)

결과: 따뜻하고 약간 탈색된 느낌. 과도한 채도 금지.
```

## 이미지 배치 규칙

### 크기 분류 (12-column 기준)

| 크기 | Column Span | 높이 기준 | 용도 |
|------|-------------|-----------|------|
| **Spot** | 3~4 col | 2~3 모듈 | 인물 초상, 디테일 |
| **Quarter** | 6 col | 3~4 모듈 | 보조 이미지 |
| **Half** | 6 col | 4~6 모듈 | 본문 중간 삽화 |
| **Hero** | 12 col (전폭) | 6~8 모듈 | 섹션 오프너, 핵심 이미지 |
| **Full Bleed** | Trim 전체 | 전체 페이지 | 사진 에세이, 표지 |

### 비율 규칙

```
- 가로 사진: 3:2 또는 16:10 (MHJ 시그니처)
- 세로 사진: 2:3 또는 3:4
- 정사각: 1:1
- 자유 크롭 허용하되, 위 비율 중 하나에 가깝게
- 사진 테두리: 없음이 기본. 필요 시 #EDE9E3 1px
```

### 캡션 규칙

```
- 위치: 사진 바로 아래, 좌측 정렬
- 서체: Inter 7.5pt, #9B9590 (Warm Gray)
- 최대 2줄
- 포맷: "설명 텍스트. 촬영: [photographer]"
- 풀 블리드 사진: 캡션은 다음 페이지 상단에 배치
```

### 가족 사진 원칙 (절대 규칙)

```
⚠️ 이 규칙은 매거진에서도 예외 없이 적용:
- 어른: 측면 또는 뒷모습만
- 아이들: 멀리서 또는 옆모습만
- 아이 실명 절대 금지 (Min, Hyun, Jin만 사용)
- 메타데이터에 실명 포함 금지
```

# 6. 12개 마스터 템플릿

## 템플릿 설계 철학

> "Content dictates layout, not the other way around."
> 콘텐츠가 레이아웃을 결정한다. 그 반대가 아니다.

각 템플릿은 특정 콘텐츠 유형에 최적화되어 있다. 편집자는 콘텐츠 성격에 맞는 템플릿을 선택하고, 그 안에서 사진/텍스트/색상을 조정한다.

## 템플릿 목록

### A. 오프너 & 풀 임팩트 (4종)

#### T01. COVER (표지)
```
┌─────────────────────┐
│                     │
│    [FULL BLEED      │
│     PHOTO]          │
│                     │
│  ┌───────────────┐  │
│  │ MHJ           │  │  ← Masthead (고정 위치)
│  │               │  │
│  │ COVER TITLE   │  │  ← 하단 1/3
│  │ subtitle      │  │
│  │ ISSUE · DATE  │  │
│  └───────────────┘  │
└─────────────────────┘

- 풀 블리드 사진 필수
- Masthead 위치: 상단 18mm 아래, 좌측 정렬
- 제목 영역: 하단 1/3, 반투명 오버레이 가능
- 변형: 사진 없는 타이포그래피 표지 (특별호용)
```

#### T02. SECTION OPENER (섹션 오프너)
```
┌─────────────────────┐
│                     │
│    [HERO PHOTO]     │
│    12-col 전폭       │
│    높이: 5~6 모듈    │
│                     │
├─────────────────────┤
│                     │
│  KICKER             │  ← Inter 7pt uppercase
│  SECTION            │  ← Playfair 36-48pt
│  TITLE              │
│                     │
│  Standfirst text    │  ← Inter 12pt italic, 8col
│  2~3줄 리드          │
│                     │
│  By Yussi · 5 min  │  ← Byline
│                     │
└─────────────────────┘

- 기사의 첫 페이지 (또는 첫 스프레드)
- 시각적 임팩트 최대화
- 변형: 사진 없이 타이포그래피만으로 구성 가능
```

#### T03. PHOTO FULL (전면 사진)
```
┌─────────────────────┐
│                     │
│                     │
│   [FULL BLEED       │
│    PHOTO]           │
│                     │
```

# 7. 이슈 구조 (Flatplan)

## Flatplan이란

Flatplan(이슈 맵)은 매거진의 페이지 배치 계획표다. 모든 프로페셔널 매거진은 콘텐츠 제작 전에 flatplan을 먼저 만든다.

## MHJ 표준 이슈 구조

MHJ 매거진은 **24페이지**를 기본으로 한다 (4의 배수 규칙 준수, 인쇄 시 필수).

```
Page 01: COVER (T01)
Page 02: Inside Front — Editor's Note or Ad
Page 03: CONTENTS (T11)
Page 04-05: OPENING FEATURE spread (T02 + T07)
Page 06: Pull Quote or Transition (T12 or T04)
Page 07-10: FEATURE STORY 1 (T02→T07→T06→T05)
Page 11: PHOTO FULL (T03) — 호흡 페이지
Page 12-15: FEATURE STORY 2 (T04→T08→T06→T09)
Page 16-17: PHOTO SPREAD (T10)
Page 18-21: FEATURE STORY 3 또는 SHORT PIECES (T07, T08, T06)
Page 22: DIRECTORY / Contributors (T11)
Page 23: Closing — Next Issue Preview
Page 24: Back Cover
```

### 리듬 원칙

```
1. 빠름 → 느림 → 빠름: 정보 밀도가 높은 페이지 → 여백 많은 호흡 페이지 → 다시 밀도
2. 매 4~5페이지마다 시각적 "쉼표": 풀 사진(T03), 인용구(T12), 또는 타이틀 카드(T04)
3. 스프레드 단위로 생각: 항상 좌우 페이지를 한 쌍으로 디자인
4. 사진 크기 변화: Small → Big → Small 리듬
```

### 페이지 수 변형

| 이슈 크기 | 페이지 수 | 기사 수 | 용도 |
|----------|----------|---------|------|
| **Mini** | 16p | 2~3편 | 특별 부록, 시즌 미니 |
| **Standard** | 24p | 3~4편 | 월간/분기 기본 |
| **Extended** | 32p | 5~6편 | 특별호, 연말호 |

# 10. 디지털 → 인쇄 전환 규격

## 현재는 디지털 퍼스트

MHJ 매거진은 **웹 뷰어에서 소비되는 디지털 매거진**이 1차 형태다. 하지만 모든 디자인 결정은 인쇄 전환을 전제로 한다.

## 디지털 렌더링 규격

| 항목 | 값 | 비고 |
|------|-----|------|
| 페이지 비율 | 42:55 (210:275) | aspect-ratio CSS |
| 뷰어 최대 너비 | 620px | 데스크톱 |
| 뷰어 최소 너비 | 320px | 모바일 |
| 렌더링 해상도 | 2x (Retina 대응) | 이미지 최소 1240px wide |
| 폰트 렌더링 | antialiased | webkit-font-smoothing |
| 색상 모드 | sRGB | 웹 표준 |

## 인쇄 전환 시 체크리스트

```
Phase 4(장기)에서 인쇄 출판 시:
☐ 색상 모드: sRGB → CMYK 변환
☐ 해상도: 모든 이미지 300dpi 이상 확인
☐ Bleed: 사방 3mm 추가
☐ 서체: 웹폰트 → OTF/TTF 임베딩
☐ Rich black: 인쇄용 검정 = C:40 M:30 Y:30 K:100
☐ 바코드/ISBN: 뒷표지에 배치
☐ 제본: 24p 이하 중철(saddle stitch), 32p 이상 무선(perfect bind)
☐ 용지: 내지 120~150gsm 무광, 표지 250~300gsm 코팅
```

# 11. 서가 & 뷰어 UI 규격

## 매거진 서가 (/magazine)

### 인터랙션 흐름

```
서가 진입
  → 매거진들이 세로로 꽂혀있는 서가 UI
  → 커버가 살짝 기울어져 보이는 3D 효과 (subtle)
  → 호버: 매거진이 살짝 빠져나오는 애니메이션 (translateY -8px, scale 1.02)
  → 클릭: 매거진 상세 페이지로 전환

매거진 상세 (/magazine/[id])
  → 표지 히어로 (큰 표지 이미지, 이슈 정보)
  → 내지 그리드 (기사 목록, 각 기사 대표 이미지 + 제목)
  → 기사 클릭 → 뷰어 모달 열림

뷰어 모달
  → 고정 비율 (42:55) 페이지
  → 좌우 넘김 (화살표 + 스와이프)
  → 페이지 넘버 표시
  → 닫기 버튼
  → 이미지 클릭 → 라이트박스
```

### 서가 카드 규격

```
커버 카드:
- 비율: 3:4 (현행 유지, 서가용 썸네일)
- 최소 너비: 180px
- 최대 너비: 280px
- 그림자: 0 4px 16px rgba(0,0,0,0.08)
- Radius: 0px (매거진은 직각)
- 호버: translateY(-8px) + shadow 강화
```

### 뷰어 규격

```
모달 배경: rgba(0,0,0,0.85)
페이지 컨테이너: aspect-ratio 42/55, max-width 620px
```

```
☐ 모든 텍스트가 Live Area(177×237mm) 안에 있는가?
```

# 부록 A: 용어 사전

| 용어 | 정의 |
|------|------|
| **Trim Size** | 최종 재단 크기 (210×275mm) |
| **Bleed** | Trim 바깥으로 확장되는 인쇄 여유 (3mm) |
| **Live Area** | 안전 영역. 모든 필수 콘텐츠가 이 안에 (177×237mm) |
| **Gutter** | 제본 쪽 여백 (18mm). 텍스트 배치 금지 |
| **Baseline Grid** | 본문 줄이 정렬되는 수평 기준선 (14pt 간격) |
| **Flatplan** | 이슈 전체 페이지 배치 계획 |
| **Masthead** | 매거진 제호 (MHJ 로고) |
| **Kicker** | 기사 제목 위의 카테고리 라벨 |
| **Standfirst** | 제목 아래 리드 문장 (2~3줄 요약) |
| **Byline** | 저자 표기 |
| **Pull Quote** | 본문에서 뽑아낸 인용구 (시각적 강조) |
| **Drop Cap** | 첫 글자를 크게 처리하는 장식 기법 |
| **Running Header** | 페이지 상단의 섹션/이슈 정보 |
| **Folio** | 페이지 번호 |
| **Spread** | 좌우 두 페이지를 한 쌍으로 본 것 |
| **Signature** | 인쇄 시 한 장의 큰 종이에 인쇄되는 페이지 묶음 (보통 8 또는 16p) |
